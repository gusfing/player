# Shrazen WordPress Plugin

Contains the Shrazen Player WordPress plugin for easy installation.

## Files

```
wordpress-plugin/
└── shrazen-player/
    ├── shrazen-player.php   # Main plugin file
    └── README.md            # Installation instructions
```

## Quick Install

1. Upload the `shrazen-player` folder to `/wp-content/plugins/`
2. Activate in WordPress Plugins menu
3. Configure in Settings > Shrazen Player

For more details, see [shrazen-player/README.md](shrazen-player/README.md)
